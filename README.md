# Node and Sprout
---

## Infrastructure
---
- Microcontroller: ESP32
- Back-end: NodeJS server using Express.js framework
- Database: PostgreSQL
- Front-end: HTML5 & EJS templating, CSS3, ECMAScript 2024

## What is Node and Sprout
---
- A full-stack web app that interactively updates a dashboard for an irrigation system
- Easily manage your irrigation to water where you want, when you want to, no matter the scale

## This Set-up
---
- Microcontroller device programmed to run a watering schedule daily between certain time periods
- Power loss functionaility that automatically continues the schedule when power is resotred
